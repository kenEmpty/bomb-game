# 💣 ボムサバイバル

爆弾で地形を破壊しながら最後の1人を目指す、2〜4人対戦のターン制サバイバルゲーム。

- HTML / CSS / JavaScript のみ・外部サーバー不要
- スマホ（Android Chrome）タップ操作に最適化、PCブラウザにも対応
- CPU対戦（Easy / Normal / Hard）、効果音・BGM（Web Audio APIで合成）

## 遊び方

ブラウザで `index.html` を開くだけ。公開版は GitHub Pages から。

各ターン：①爆弾を投げる（8方向・射程1〜2）→ ②3歩移動 → ③開始マスが破壊される。
最後まで生き残ったプレイヤーの勝ち。

## バランス調整

`js/config.js` の定数（移動歩数・爆弾射程・障害物数・マップサイズ・CPU思考時間・アニメ速度）で調整できます。
